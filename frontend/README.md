# Mocky Pops Frontend

Static frontend for Mocky Pops customer website, admin panel, delivery dashboard, franchise pages, gift cards, loyalty, events and AI chatbot.

Open `index.html` in a browser for the customer website.

Run the backend first from the project root:

```powershell
.\start-backend.bat
```

Main pages:

- `index.html`
- `admin-login.html`
- `admin.html`
- `login.html`
- `register.html`
- `profile.html`
- `cart.html`
- `checkout.html`
- `orders.html`
- `delivery.html`
- `franchise.html`
- `franchise-owner.html`
- `gift-card.html`
- `loyalty.html`
- `events.html`
